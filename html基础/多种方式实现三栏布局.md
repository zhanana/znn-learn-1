## flex 布局(建议用这种，大势所趋)

```javascript
.container{
            height: 400px;
            display: flex;
        }
        .left{
            width: 200px;
            background-color:  goldenrod;
        }
        .center{
            flex:1;
            background-color:  rgb(41, 201, 36);
        }
        .right{
            width: 200px;
            background-color:  rgb(36, 195, 201);
        }
    <div class="container">
        <div class="left">left</div>
        <div class="center">center</div>
        <div class="right">right</div>
    </div>
```

## 利用 calc 计算出中间快所占据的大小，注意注意，写 div 的时候一定不能换行

```javascript
.container{
            height: 400px;
            background-color: hotpink;
        }
        .container > div{
            display: inline-block;
            height: 100%;
            color: indigo;
        }
        .container > .left{
            width: 200px;
            background-color: rgb(231, 170, 170);
        }
        .container > .center{
            width: calc(100% - 400px);
            background-color: lightsalmon;
        }
        .container > .right{
            width: 200px;
            background-color: rgb(235, 26, 26);
        }
    <div class="container">
        <div class="left">left</div><div class="center">center</div><div class="right">right</div>
    </div>
```

```javascript
 .container{

        }
        .container div{
            height: 100%;
        }
        .container .left{
            width: 200px;
            float: left;
            background-color: rgb(199, 220, 233);
        }
        .container .right{

        }
        .container .center{
            display: flex;
            background-color: rgb(76, 128, 161);
        }
    <div class="container">
        <div class="left">left</div>
        <div class="right">right</div>
        <div class="center">center</div>
    </div>
```
